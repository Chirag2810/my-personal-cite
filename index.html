Sign Language recognition System

The sign language recognition system is a system for people with speech and hearing incapability. This system helps them to communicate easily with the common people around them. The working of the system is as such, the speech and hearing impaired people just have to use hand gestures(sign language) to communicate, which would be captured in the camera and the equivalent letters to the gestures would be displayed on the screen. The system is user friendly and every common man need not learn sign language to communicate with the specially challenged people.

The components and softwares used were:
  • Raspberry Pi 3 B+
  • Raspberry Camera module
  • Edge Impulse software
